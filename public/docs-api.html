Internal API Console

An interactive, Swagger-style catalog of CodeCanvas's internal API surface.

Read this first. CodeCanvas is a local desktop app, not a web service. The operations below are internal — IPC channels, dependency-injection services, commands, the Claude agent-host protocol, the penpal design bridge, HTML write-back, and the Rust CLI's RPC. They are modeled as OpenAPI purely so you can browse them with a familiar interface. The host, paths, and the "Try it out" button are illustrative; there is no server to call.
